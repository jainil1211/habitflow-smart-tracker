# 🌌 HabitFlow – Smart Habit Tracking System

HabitFlow is a premium full-stack habit tracking application built using the MERN stack.  
It helps users build consistency through streak tracking, leaderboard competition, calendar analytics, and smart reminders — all wrapped in a modern “Space Black” UI inspired by premium SaaS products.

---

## 🚀 Features

### 🔐 Authentication
- User Registration & Login
- JWT-based Authentication
- Password hashing with bcrypt

### ✅ Habit Management
- Create, update, and delete habits
- Track daily completion
- Habit streak system

### 🔥 Gamification
- XP / Points system
- Level progression
- Weekly leaderboard

### 📅 Calendar Analytics
- Monthly calendar view
- Visual performance tracking
- Completed vs missed day insights

### ⏰ Smart Reminders
- Browser-based habit reminders
- Reminder scheduling system

### 🎨 Premium UI
- Space Black theme
- Responsive dashboard
- Glassmorphism-inspired design
- Modern SaaS-style layout

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/jainil1211/habitflow-smart-tracker.git
```

---

### 2. Backend Setup

```bash
cd backend
npm install
```

Create `.env`

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret
```

Run backend:

```bash
npm run dev
```

---

### 3. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
